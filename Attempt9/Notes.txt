Design goals:
Fast
Easy to read
A true compiled language
Strong typing
Structural typing
Expression based
Curly brace / semicolon blocks

Binary operators should use words, not symbols
    “and”, “or”, “not”


Program					(Constant)*

Constant				VAR IDENTIFIER EQUALS Expression

Expression				Block 
						| IfExpression 
						| AddExpression

AddExpression			MultiplyExpression
						(ADD | SUBTRACT MultiplyExpression)*

MultiplyExpression		Atom 
						(MULTIPLY | DIVIDE Atom)*

IfExpression			IF Expression 
						THEN Expression 
						ELSE Expression

Block					OPENBRACE 
						Expression (SEMICOLON Expression)* 
						CLOSEBRACE

Atom					INT64
						| FLOAT64
						| OPENPAREN Expression CLOSEPAREN
